# 主应用详解

主应用是整个微前端架构的核心，负责整体布局、路由分发以及子应用的注册和管理。

## 架构概述

主应用采用 Vue 3 + Vite + TypeScript 技术栈，通过 Qiankun 微前端框架实现子应用的集成。

### 核心功能

1. **布局管理**：提供统一的页面布局，包括头部、侧边栏和内容区域
2. **路由分发**：负责主应用路由和子应用路由的分发
3. **子应用管理**：注册、加载和卸载子应用
4. **应用间通信**：提供主应用与子应用之间的通信机制

## 项目结构

```
src/
├── components/          # 公共组件
├── layout/              # 布局组件
├── micro/               # 微前端相关配置
├── router/              # 路由配置
├── shared/              # 共享工具和配置
├── stores/              # 状态管理
├── styles/              # 样式文件
├── types/               # TypeScript 类型定义
├── utils/               # 工具函数
├── views/               # 页面视图
├── App.vue             # 根组件
└── main.ts             # 入口文件
```
